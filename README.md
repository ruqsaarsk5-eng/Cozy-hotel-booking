# Cozy-hotel-booking🌙 
Flutter Web App

A simple and clean productivity web app built using Flutter, inspired by a cozy UI design.
This app includes a music player input, brainstorming notes section, and a contact form — all in one place.

---

🚀 Features

- 🎵 Music Player UI
  
  - Paste streaming URL
  - Load music (UI ready)

- 🧠 Brainstorm / Notes Section
  
  - Write ideas easily
  - Clear notes instantly
  - Export option (UI ready)

- 📩 Contact Form
  
  - Name, Email, Message fields
  - Simple clean layout

- 🌐 Responsive Web UI
  
  - Works on Chrome (Flutter Web)
  - Minimal and modern design

---

🛠️ Tech Stack

- Flutter (Web)
- Dart
- Material UI

▶️ Getting Started

1. Clone the Repository

git clone https://cozy-creponne-78b570.netlify.app/
cd cozy_app

---

2. Install Dependencies

flutter pub get

---

3. Run the App (Web)

flutter run -d chrome
